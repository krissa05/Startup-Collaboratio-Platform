# Startup Collaboration Platform

A web-based platform connecting Investors, Developers, and Founders through a role-based system.

## Features
- Idea sharing
- Collaboration system
- Role-based access
- User interaction

## Tech Stack
- Backend: Flask (Python)
- Database: Oracle DB, SQL
- Frontend: HTML, CSS, JavaScript

## Description
Implemented DBMS concepts like triggers, procedures, and sequences for efficient data handling and system performance.
